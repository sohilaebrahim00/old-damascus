1. Upload this folder to a new GitHub repository.
2. Do not upload any .env.local file.
3. In Netlify, choose:
   Add new project -> Import an existing project -> GitHub.
4. Select the repository.
5. Use:
   Build command: npm run build
6. Leave the publish directory auto-detected.
7. Add the required environment variables in:
   Site configuration -> Environment variables.
8. Deploy the site.
9. Test menu, authentication, cart, API routes, and map.
10. Keep live checkout disabled until approved.
